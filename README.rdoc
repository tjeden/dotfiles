git config for everyday work

== Pre

Make copy of your old files

  cp -r .vim .vim_cold
  cp .vimrc vimrc_old
  cp .viminfo .viminfo_old
  
== Installation

Make symlinks to git repo

  ln -s code/dotfiles/.vim .vim
  ln -s code/dotfiles/.vimrc .vimrc
  ln -s code/dotfiles/.viminfo .viminfo

Vundle plugins:

  vim
  :BundleInstall
